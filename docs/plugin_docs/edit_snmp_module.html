inspur.sm.edit_snmp – Set snmp.¶

Note

This plugin is part of the inspur.sm collection.

To install it use: ansible-galaxy collection install inspur.sm.

To use it in a playbook, specify: inspur.sm.edit_snmp.

New in version 0.1.0: of inspur.sm

Synopsis¶

  • Set snmp on Inspur server.

Examples¶

- name: Snmp test
  hosts: ism
  connection: local
  gather_facts: no
  vars:
    ism:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Set snmp get/set"
    inspur.sm.edit_snmp:
      community: "test"
      v3username: "Inspur"
      provider: "{{ ism }}"
